AI-Powered Chrome Extension

Extract and analyze web page data with the power of Groq AI

Overview

This Chrome Extension captures content from any webpage, leverages Groq AI for processing, and generates a downloadable .txt file. This is especially useful for product managers, consultants, or researchers who want to quickly extract and digest webpage content.

Say goodbye to manually going through long articles. Get intelligent summaries delivered instantly!

Try It Now!

  1. Clone this repository and install the packages: 
    git clone https://github.com/your-username/chrome-ext-webpage-data.git
    npm i
  2. Open Google Chrome and go to chrome://extensions/
  3. Enable Developer mode
  4. Click "Load unpacked" and select the cloned project folder
  5. Open any webpage
  6. Click the Extensions icon and choose "My Chrome Extension"
  7. Click the "Get Info!!!!" button

Troubleshoot

If you're facing issues with the extension not working or the dist folder missing, try the following:

  1. Remove the old dist folder: 
    rm -rf dist/
  2. Rebuild the extension bundle using Webpack: 
    npx webpack --config webpack.config.js
  3. Ensure you are loading the freshly built dist folder in chrome://extensions/.
  4. If problems persist, check the browser console and terminal for errors.
